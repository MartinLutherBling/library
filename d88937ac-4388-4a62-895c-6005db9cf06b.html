Description

Stream Avatars is an interactive overlay for live broadcasters on Twitch, Facebook, Youtube, Trovo, or Dlive. Your viewers can knock each other off the screen with punches, bombs, or a duel show match where the winner takes all!



Unleash Your Creativity!

You can create your own avatars, equipment, and name tags to fit the theme of your stream.
If creativity is not your strong suit, download some community-made Avatars from the Steam workshop!

Currency Integrations

Mixitup, Streamlabs, & Stream Elements

Disclaimers

  • Do you use a dual PC setup? - If you do, you might need to get Stream Avatars on a secondary steam account for the streaming machine instead of your primary steam account! Family Sharing is not possible for Software on steam*
  • Newer Mac OSX versions cannot do Game Captures, you must use window capture with Chroma key filter to erase the background.
